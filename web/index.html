// EXPORT MENU //

Export to FreeCAD Downloads a .FCStd file openable in FreeCAD. Generates 3D geometry for walls, framing, and apertures directly in the browser — no server required.
Export to JSON (Legacy / Developer) Downloads the layout as a v2 JSON file for use with the compiler pipeline. Intended for developers and advanced users feeding into external tooling.
Save Your Work Downloads your layout as a JSON file you can reload later. Use Load in the sidebar to restore a saved file.
Load Previous Design Load a previously saved JSON layout file back into the tool.
Export to IFC (Coming Soon) IFC (Industry Foundation Classes) is an open BIM standard supported by Revit, ArchiCAD, and other AEC tools. Export will be available in a future release.
Export Fab Drawings Generates per-panel SVG fabrication drawings with cut dimensions and stud layout. Downloads as a print-to-PDF build book.
Generate Build Summary A coordination packet: modules grouped by run, an in-place build map, a whole-build cut & pick list, plus one fab card per distinct design. Opens as a print-to-PDF book.
